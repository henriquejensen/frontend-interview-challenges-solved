Introduction

The goal of this program is to draw a circle inside a paralellogram with 3 points that you choose.

Program flow

You need to click in 3 differents points of the black box inside the page. After it the program will draw a paralellogram and a circle with this 3 given points.

Options

The green button will permit you to reorder some point with you wish.

You can reset the screen clicking on the red button on the top.

On the left of the black box you can see the coordinates of the each point choosed and the coordinates of the last point of the paralellogram, the paralellogram area and the circle area.
